## Project Topic

**07 – Development of a Code Verifier for Code Security**

This project focuses on the development of a tool capable of verifying the security of program code. The objective is to analyze source code and detect potential vulnerabilities that could be exploited by attackers.

The analyzer checks for common security vulnerabilities such as:

- SQL Injection
- Cross-Site Scripting (XSS)
- Buffer Overflow

The solution provides information about the detected vulnerabilities and includes a **vulnerability assessment** of the analyzed code.

### Project Requirements

- Analyze program code for potential security vulnerabilities.
- Detect at least **two different types of vulnerabilities**.
- Provide **clear information about detected issues**.
- Include a **basic vulnerability assessment** indicating the severity or risk level of the detected problems.
